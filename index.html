Sinkronisasi Cloud Real-Time (Google Drive)

Hubungkan buku tamu ini secara otomatis ke Google Sheets milik Dinas Anda di Google Drive.

Belum Terhubung

URL Web App hasil deploy Google Apps Script Anda tersimpan secara default di dalam kode aplikasi.

Panduan Pemasangan Google Drive

  1. Buka Google Drive, buat Google Sheets baru dengan nama "E-Tamu UKPBJ".
  2. Di menu atas Google Sheets, buka Ekstensi > Apps Script.
  3. Salin kode Apps Script (klik tombol hitam di bawah) lalu paste-kan di editor Google Apps Script tersebut.
  4. Klik Deploy > Penerapan Baru. Pilih jenis: Aplikasi Web.
  5. Ubah Pengaturan Akses ke: Siapa Saja (Anyone).
  6. Deploy, berikan izin Google, lalu salin URL Aplikasi Web-nya ke kolom kiri.
